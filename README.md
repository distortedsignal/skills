# Skills
A repo for LLM skills.

| Skill | Description |
| ----- | ----------- |
| `test-all-makefile-targets` | Test all the makefile targets in the current directory and in any subdirectories |
| `create-makefile-callgraph` | Create a callgraph of the targets in all makefiles in a project |
| `document-code-in-readme-files` | Read code in subdirectories, and derive README.md documents from the code |
